Providenciado


Cadastrar uma entidade no JSON Server

As dependências necessárias para o bom funcionamento da aplicação estão no arquivo "package.json"

Pode-se verificar na linha 22 do arquivo "package.json" a dependência "json-server": "^0.17.0"

Para funcionamento correto deve-se acessar via terminal a pasta "src/app/dados" e executar o comando "json-server --watch db.json"

O conteúdo deve estar disponível no caminho "http://localhost:3000/conteudoNotas". Se necessário, atualizar o endereço na variável "url" do arquivo "src\app\services\service-notas-mentais.service.ts"

É possível verificar a utilização do JSON server arquivos "src\app\services\service-notas-mentais.service.ts", "src\app\nova-nota-mental\nova-nota-mental.component.ts", "src\app\exibicao-nota-mental\exibicao-nota-mental.component.ts"
